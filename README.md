# KiVO - Your Ultimate Student Hub

KiVO is an all-in-one student application designed to help you organize your academic life, manage your well-being, and stay motivated.

## Features

*   **Interactive Calendar:** Manage homework, events, and deadlines.
*   **AI Study Tools:** Flashcards, quick references, and AI-powered document analysis.
*   **Virtual Pet Companion:** Care for your digital pet and earn rewards as you progress.
*   **Wellness Hub:** Track your mood and receive personalized wellness suggestions.
*   **MoneyFlow:** Track expenses and manage budgets.
*   **Scrapbook:** Capture and cherish your memories.
*   **Achievements:** Earn badges for your progress and accomplishments.

## How to Run Locally

1.  Clone the repository: `git clone [YOUR_REPO_URL]`
2.  Navigate to the project directory: `cd kivo`
3.  Install dependencies: `npm install` (or `yarn install`)
4.  Start the development server: `npm start` (or `yarn start`)
5.  Open your browser to `http://localhost:3000` (or the address provided in your terminal).
